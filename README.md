# Cyberdeck Dashboard
A dashboard for a CyberDeck (Made for my friend Schnauz)

## This is a very simple GUI for a secondary screen using an external device (like an Raspberry Pi)
Basically, all it does is integrate a bunch of functionality into a GUI that can be easily controled with a touchscreen (besides the note widget, that requires a keyboard)

Don't expect anything fancy at all here, I'm just using this to learn more about python (specially GUIs) and to help my friend built the Cyberdeck project she wants :)

# How does it Work?
### Theres two main parts:
## 1- PC Server (For System Monitoring)
Install the pc_server.py program on your PC only. On Linux (My OS) it works out of the box, other OS may need extra configurations.
## 2- Main Code (For a mobile system, like the Raspberry Pi)
Install eberything else, then configure these things:
1- pc_ip on theme.py, set it to your main PC local ip address
2- change the .env values for SPOTIFY_CLIENT_ID and SPOTIFY_CLIENT_SECRET to their respective values on your own Spotify application.
3- change the SPOTIFY_REFRESH_TOKEN value to the value generated by the url above it when you log your own account
4- change the weather_city value in theme.py to your city
